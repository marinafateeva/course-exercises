VAR

The var keyword was used in all JavaScript code from 1995 to 2015 to declare a value that can be changed. The var keyword should only be used in code written for older browsers.


LET

The let and const keywords were added to JavaScript in 2015. LET is currently used to declate a value that can be changed. Let is used when const cannot be.


CONST

Const is used if the value or the type (arrays and objects) should not be changed.
